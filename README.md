# Warrior Arms

My boilerplate to ReactJS

## Usage

Install packages:
```sh
./warrior.sh
```

Run webpack-dev-server:
```sh
yarn start
```

Run StoryBook
```sh
yarn storybook
```

Run Tests with Jest
```sh
yarn test
```

Run Build
```sh
yarn build
```

Be Happy :smiley:
